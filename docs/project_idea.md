# Game verse

## Description
Game verse is a digital universe for gamers where users can purchase, play, review, and socially engage around games within a unified platform.

### Core Features
- User Accounts
- Game Store
- Game Library
- Reviews and Ratings
- Friends & Chat System
- Discussion Portal
- Wishlist
- Achievements
- Coin System
- Digital Payment Support 

#### Output
Database queries will be developed in Visual Studio Code and validated using pgAdmin 4. The resulting data will be transformed into interactive Power BI dashboards, enabling stakeholders and non-technical users to analyze key metrics and gain actionable insights.
